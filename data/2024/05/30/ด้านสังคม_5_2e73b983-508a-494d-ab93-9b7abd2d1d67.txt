ดีอี ร่วมงาน AWS Summit Bangkok ผลักดันการใช้เทคโนโลยี Cloud ในไทย
ดีอี ร่วมงาน AWS Summit Bangkok ผลักดันการใช้เทคโนโลยี Cloud ในไทย
วันนี้ (30 พฤษภาคม 2567) นายเศรษฐา ทวีสิน นายกรัฐมนตรี เป็นประธานกล่าวเปิดงาน AWS Summit Bangkok โดยมี นายประเสริฐ จันทรรวงทอง รัฐมนตรีว่าการกระทรวงดิจิทัลเพื่อเศรษฐกิจและสังคม (ดีอี) นายเวทางค์ พ่วงทรัพย์ รองปลัดกระทรวงดีอี นายสุทธิเกียรติ วีระกิจพานิช ที่ปรึกษารัฐมนตรีว่าการกระทรวงดีอี นายวัลลภ รุจิรากร เลขานุการรัฐมนตรีว่าการกระทรวงดีอี และผู้ที่เกี่ยวข้องเข้าร่วม จัดโดย Amazon Web Services บริษัทผู้เชี่ยวชาญด้าน Cloud Computing ณ ศูนย์การประชุมแห่งชาติสิริกิติ์
ในงานนี้นายกฯกล่าวยินดี ที่ AWS มีแผนลงทุนสร้าง Data Center ในไทย มูลค่ากว่า 190,000 ล้านบาท ภายในปี 2037 พร้อมยืนยันรัฐบาลจะสนับสนุนความร่วมมือในลักษณะ win-win เช่นนี้ และเชื่อมั่นว่าการลงทุน และการสนับสนุนอย่างต่อเนื่องของทั้ง AWS และทุกภาคส่วนในไทยจะยกระดับ Digital Literacy ให้กับคนไทย และนำไปสู่การเติบโตทางเศรษฐกิจในระยะยาว
