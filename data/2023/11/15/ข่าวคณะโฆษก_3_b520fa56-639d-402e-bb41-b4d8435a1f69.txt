พิมพ์
​โฆษกรัฐบาล เผยความสำเร็จในการดึงดูดการลงทุนของนายกรัฐมนตรี เกิดการลงทุนแล้ว 3 บริษัท AWS (Amazon Web Services), Microsoft และ Google มูลค่ารวมกว่า 3 แสนล้านบาท
​โฆษกรัฐบาล เผยความสำเร็จในการดึงดูดการลงทุนของนายกรัฐมนตรี เกิดการลงทุนแล้ว 3 บริษัท AWS (Amazon Web Services), Microsoft และ Google มูลค่ารวมกว่า 3 แสนล้านบาท
นายสัตวแพทย์ชัย วัชรงค์ โฆษกประจำสำนักนายกรัฐมนตรี เปิดเผยความสำเร็จของของนายเศรษฐา ทวีสิน นายกรัฐมนตรีและรัฐมนตรีว่าการกระทรวงการคลังตั้งแต่ช่วงต้นของการเดินทางเยือน นครซานฟรานซิสโก สหรัฐอเมริกา เนื่องจากเกิดการลงทุนในไทยจาก 3 บริษัทชั้นนำระดับโลก AWS (Amazon Web Services), Microsoft และ Google ลงทุนในไทย
โฆษกประจำสำนักนายกรัฐมนตรีกล่าวว่า นายกรัฐมนตรีมุ่งมั่นนำเสนอนโยบายของรัฐบาล และศักยภาพของไทยต่อบริษัทฯ ชั้นนำในหลายประเทศเพื่อให้เกิดการลงทุนในประเทศไทย ซึ่งเห็นเป็นผลสำเร็จในการเดินทางครั้งนี้ ดังนี้
AWS บริษัทด้าน hyperscale data center ระดับโลก ได้ลงทุนก่อสร้าง AWS Asia Pacific (Bangkok) Region สร้างผลประโยชน์ทางเศรษฐกิจในไทย เพิ่มขีดความสามารถในการแข่งขัน รวมทั้งปรับปรุงประสิทธิภาพการปฏิบัติงานของภาครัฐ และช่วยสร้างแรงงานที่มีทักษะสูงที่ประเทศไทยกำลังขาดแคลน
บริษัท Microsoft ได้ลงนามในบันทึกความเข้าใจ (MOU) กับรัฐบาลไทย เป็นโอกาสพัฒนาเทคโนโลยีดิจิทัล เพิ่มขีดความสามารถในการแข่งขันทางดิจิทัลของประเทศไทย รวมทั้งช่วยปรับปรุงการให้บริการสาธารณะผ่านการเปลี่ยนแปลงบริการรัฐบาลอิเล็กทรอนิกส์ พร้อมกันนี้ ไทยได้เตรียมใช้กลไกใหม่ที่เรียกว่า Utility Green Tariff (UGT) เพื่อรองรับการใช้พลังงานสะอาด ทำให้บริษัทได้ใช้พลังงานหมุนเวียนผ่านโครงข่ายไฟฟ้าของรัฐในอัตราค่าบริการที่เหมาะสม โปร่งใส และเป็นธรรมซึ่งเป็นปัจจัยที่ Microsoft ให้ความสำคัญ
รวมทั้ง บริษัท Google ได้ลงนาม MOU ระหว่าง Google กับกระทรวงดิจิทัลเพื่อเศรษฐกิจและสังคม (MDES) ถือเป็นความสำเร็จเพื่อไปสู่ความก้าวหน้าทางเทคโนโลยี เพิ่มขีดความสามารถในการแข่งขันทางดิจิทัล และเชื่อมั่นว่าจะมีส่วนสำคัญช่วยให้ประเทศเติบโตทางเศรษฐกิจ ซึ่ง Google ได้พิจารณาไทยเป็นประเทศที่มีศักยภาพสำหรับเป็นสถานที่ตั้งศูนย์ข้อมูล (Data Center) เพิ่มเติมเป็นแห่งที่ 4 ในบริเวณภูมิภาคเอเชีย-แปซิฟิก และเป็นแห่งที่ 11 ของโลก
“นายกรัฐมนตรีถือว่านี้เป็นเพียงช่วงเริ่มต้นของความสำเร็จ รัฐบาลยังดำเนินการต่อเนื่องในการเชิญชวนบริษัทชั้นนำร่วมลงทุนในไทย เปิดศักยภาพของประเทศ กำหนดนโยบายที่ดึงดูดการลงทุน เพิ่มความเชื่อมั่นให้นักลงทุนทั่วโลก โดยนายกรัฐมนตรีเชื่อว่าด้วยศักยภาพ และปัจจัยสนับสนุนของไทยทั้งด้าน ภูมิศาสตร์ โครงสร้างพื้นฐาน ระบบสาธารณสุข ระบบการศึกษา ทำให้ไทยมีจุดเด่นน่าสนใจไม่แพ้ใครในภูมิภาค” นายสัตวแพทย์ชัย กล่าว
