รัฐบาลไทย-ข่าวทำเนียบรัฐบาล-“กรุงไทย” ผนึก “ฟิเดลิตี้” เปิดกองทุน Krungthai World Class Series ดีเดย์ 2-11 ส.ค.66


วันพุธที่ 26 กรกฎาคม 2566
26/07/2566
พิมพ์
“กรุงไทย” ผนึก “ฟิเดลิตี้” เปิดกองทุน Krungthai World Class Series ดีเดย์ 2-11 ส.ค.66
“ธนาคารกรุงไทย” จับมือ “ฟิเดลิตี้” เปิดกองทุนใหม่ “Krungthai World Class Series” รวม 3 กองทุน แยกตามระดับความเสี่ยงและผลตอบแทนที่คาดหวัง ผ่าน Krungthai NEXT และสาขาทั่วประเทศ ดีเดย์  2-11 ส.ค.66
“ธนาคารกรุงไทย” จับมือ “ฟิเดลิตี้” ต่อยอดความร่วมมือ สร้างประสบการณ์ลงทุนเหนือระดับให้ลูกค้า เปิดกองทุนใหม่ “Krungthai World Class Series” รวม 3 กองทุน แยกตามระดับความเสี่ยงและผลตอบแทนที่คาดหวัง มุ่งจับเทรนด์การเติบโตในสินทรัพย์คุณภาพดีทั่วโลก โดยผู้เชี่ยวชาญการลงทุนระดับโลก จากฟิเดลิตี้ ลงทุนง่ายเริ่มต้น 1 บาท ผ่าน Krungthai NEXT และสาขาทั่วประเทศ ดีเดย์  2-11 ส.ค.66
นายรวินทร์ บุญญานุสาสน์ รองกรรมการผู้จัดการใหญ่ สายงานธุรกิจตลาดเงินตลาดทุน ธนาคารกรุงไทย เปิดเผยว่า  ธนาคารกรุงไทย ในฐานะธนาคารพาณิชย์ชั้นนำของประเทศ มุ่งมั่นพัฒนาผลิตภัณฑ์การออมและการลงทุน เพื่อสนับสนุนการสร้างผลตอบแทนจากการลงทุนในทุกจังหวะของตลาด เดินหน้าต่อยอดความร่วมมือกับฟิเดลิตี้ อินเตอร์เนชันแนล (Fidelity International) และ บริษัทหลักทรัพย์จัดการกองทุนกรุงไทย (KTAM)    ยกระดับบริการบริหารความมั่งคั่ง เพื่อสร้างประสบการณ์การลงทุนเหนือระดับให้กับลูกค้าธนาคาร ด้วยการออกและเสนอขายกองทุน Krungthai World Class Series ชูจุดเด่น เรื่องการบริหารจัดการโดยผู้เชี่ยวชาญด้านการลงทุนระดับโลกจากฟิเดลิตี้  เน้นบริหารกองทุนในเชิงรุก (Active Management) กระจายการลงทุนในสินทรัพย์คุณภาพดีทั่วโลก ปรับพอร์ตการลงทุนรองรับทุกสภาวะตลาดทันท่วงที และมีกลยุทธ์การบริหารจัดการความเสี่ยงอย่างมืออาชีพ
ธนาคารประกาศความร่วมมือเป็นพันธมิตรธุรกิจกับฟิเดลิตี้ บริษัทจัดการลงทุนระดับโลก โดดเด่นด้วยประสบการณ์ทำธุรกิจมากกว่า 50 ปี ใน 25 ประเทศ ให้คำปรึกษาการวางแผนเกษียณให้กับลูกค้ากว่า 2.8 ล้านคนทั่วโลก บริหารสินทรัพย์รวมมูลค่ากว่า 7.28 แสนล้านดอลลาร์สหรัฐ   เพื่อยกระดับบริการด้านการลงทุนของธนาคารแบบครบวงจร นำเสนอโซลูชันทางการเงินที่ดีที่สุดให้กับลูกค้าธนาคาร ทั้งการเป็นที่ปรึกษาการลงทุนส่วนบุคคล การแนะนำการจัดสรรพอร์ตเงินลงทุน กลยุทธ์การจับจังหวะตลาดเพื่อเข้าลงทุน การพัฒนาผลิตภัณฑ์ลงทุนที่หลากหลายไร้พรมแดน  และบริการด้านบทวิเคราะห์
กองทุน Krungthai World Class Series มีทั้งหมด  3 กองทุน ตามเป้าหมายการลงทุน และความเสี่ยงที่เหมาะสมสำหรับลูกค้า ดังนี้
1. Krungthai World Class Defensive (KTWC-Defensive) ความเสี่ยงต่ำ ลงทุนในตราสารหนี้ ทั้งตราสารหนี้ไทยและตราสารหนี้ต่างประเทศ ในสัดส่วนไม่ต่ำกว่า 70% เหมาะสำหรับลูกค้าที่ต้องการผลตอบแทนการลงทุนที่ชนะเงินเฟ้อ และยอมรับความผันผวนได้ไม่มาก
2. Krungthai World Class Moderate  (KTWC-Moderate) ลงทุนในตราสารทุนตั้งแต่ 30-70% เหมาะสำหรับลูกค้าที่รับความผันผวนได้มากขึ้น เน้นการเติบโตของเงินลงทุน และอยากกระจายการลงทุน
3. Krungthai World Class Growth  (KTWC-Growth) ลงทุนในตราสารทุนไม่ต่ำกว่า 60% เหมาะสำหรับลูกค้าที่ต้องการผลตอบแทนสูง และรับความผันผวนได้
กองทุน Krungthai World Class Series เสนอขายหน่วยลงทุนครั้งแรก (IPO) ระหว่างวันที่ 2-11 สิงหาคม 2566  ผ่านแอปพลิเคชัน Krungthai NEXT และสาขาธนาคารกรุงไทยทั่วประเทศ ด้วยเงินลงทุนขั้นต่ำเพียง 1 บาท พิเศษ สำหรับลูกค้าที่มียอดซื้อ-สับเปลี่ยนเข้ากองทุน Krungthai World Class Series ระหว่างวันที่ 2-11 สิงหาคม 2566 ตั้งแต่ 50,000 บาทขึ้นไป รับ Gift Voucher Central เริ่มต้น 100 บาท และสูงสุดไม่เกิน 40,000 บาท ตามยอดเงินลงทุน สอบถามข้อมูลเพิ่มเติมได้ที่ธนาคารกรุงไทยทุกสาขา
