วันเสาร์ที่ 17 พฤษภาคม 2568
17/05/2568
พิมพ์
นายกฯ แพทองธาร หนุนการขับเคลื่อนโครงการ Thai SELECT ชวนทุกภาคส่วนร่วมกิจกรรม Thai Fruits Festival  กระตุ้นการบริโภคผลไม้ภายในประเทศ
นายกฯ แพทองธาร หนุนการขับเคลื่อนโครงการ Thai SELECT ชวนทุกภาคส่วนร่วมกิจกรรม Thai Fruits Festival  กระตุ้นการบริโภคผลไม้ภายในประเทศ
วันนี้ ( 17 พฤษภาคม 2568) เวลา 12.30 น. ณ บริเวณโรงแรมมณีจันทร์รีสอร์ท จังหวัดจันทบุรี นางสาวแพทองธาร ชินวัตร นายกรัฐมนตรีและคณะ เยี่ยมชมบูธผลไม้ตามฤดูกาลและผลไม้แปรรูปและร้านอาหารที่ได้รับเครื่องหมาย Thai SELECT และร่วมกิจกรรมรณรงค์การรับประทานผลไม้ไทยตามฤดูกาล โดยมีนายมนต์สิทธิ์ ไพศาลธนวัฒน์ ผู้ว่าราชการจังหวัดจันทบุรี นางอรมน ทรัพย์ทวีธรรม อธิบดีกรมพัฒนาธุรกิจการค้า นายวิทยากร มณีเนตร อธิบดีกรมการค้าภายใน ผู้ประกอบการร้านอาหาร Thai SELECT เข้าร่วม
นายจิรายุ ห่วงทรัพย์ โฆษกประจำสำนักนายกรัฐมนตรี เปิดเผยว่า นายกรัฐมนตรีเยี่ยมชมกิจกรรมต่าง ๆ พร้อมมอบป้ายตราสัญลักษณ์ Thai SELECT แก่ร้านอาหารไทยในจังหวัดจันทบุรีจำนวน 3 รายได้แก่ 1) ครัวมณีจันทร์ ที่นำเสนอ มัสมั่นทุเรียน ยำสลัดสุมาลี และทุเรียนเชื่อม 2) ร้านจันทรโภชนา สาขามหาราช ที่นำเสนอ แกงเผ็ดเป็ดย่างเงาะ ยำมังคุด ผัดกระวานปลากระพง ส้มตำทุเรียน และ 3) ร้านรัตนะบุรี นำเสนอ ข้าวเหนียวทุเรียน หมูชะมวงที่ได้รับรางวัลชนะเลิศการประกวดอาหารพื้นบ้านอันดับ 1 ของจังหวัดจันทบุรี
นายกรัฐมนตรี กล่าวชื่นชมร้านอาหารที่ได้รับตราสัญลักษณ์ Thai SELECT ซึ่งจังหวัดจันทบุรีที่มีวิธีการปรุงอาหารให้ได้รสชาติออกมาแบบ ไทยแท้ ๆ นอกจากนี้ ยังได้ลองชิมอาหารที่นำเอาผลไม้ ของดี ของอร่อยภาคตะวันออก มาประกอบการปรุงอาหารให้ได้อาหารที่แปลกใหม่ และมีรสชาติอร่อยเป็นเอกลักษณ์ การนำเอาผลไม้มาดัดแปลง ทำเป็นอาหารประเภทต่าง ๆ จะช่วยเพิ่มมูลค่าให้แก่ผลผลิต "เชิญชวนคนไทยทานผลไม้ไทยกันเยอะๆ ซึ่งปีนี้ผลไม้ไทยออกผลผลิตมาก เพราะจากสภาพอากาศที่เอื้ออำนวยในช่วงติดดอก เลยเป็นโอกาสสำคัญที่เราจะช่วยกันส่งเสริมผลไม้ไทยอีกด้วย"นายกรัฐมนตรี กล่าว
สำหรับตราสัญลักษณ์ Thai SELECT กระทรวงพาณิชย์ ได้ดำเนินการส่งเสริมการพัฒนาศักยภาพการตลาดและความสามารถในการแข่งขันของธุรกิจร้านอาหารไทยผ่านตราสัญลักษณ์ Thai SELECT พร้อมสโลแกน "อาหารไทย ต้อง Thai SELECT" เพื่อเชื่อมโยงการค้าจากภายนอกสู่ท้องถิ่น กระจายรายได้และเสริมความเข้มแข็งให้ผู้ประกอบการในท้องถิ่น ซึ่งปัจจุบันมีร้านอาหารที่ได้รับตราสัญลักษณ์ Thai SELECT จำนวน 496 ร้านทั่วประเทศ ซึ่งช่วยสร้างโอกาสทางการตลาดและรับรองคุณภาพรสชาติไทยแท้ ซึ่งช่วยยกระดับมาตรฐานคุณภาพร้านอาหารไทยและเพิ่มโอกาสทางการตลาดด้วย
